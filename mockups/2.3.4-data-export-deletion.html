Export & Delete Data

Parents can request a session summary export or delete persisted profile data. Deletions are irreversible.

Request export
Export format: CSV or PDF. Processing time: immediate for small sessions.
Delete profile data
This removes persisted progress and profile info. Guest/session-only data is unaffected.
Deletion requires explicit confirmation: type the account name or tap a confirmation checkbox. A recovery link can be provided within 24 hours if requested immediately.
